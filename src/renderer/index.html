    Auto-record meetings (Windows)
    
          
    Nothing runs until a Zoom/Teams meeting starts — then the app boots cold, shows a notification (click it to start recording), and quits itself if you don't. It only uses memory briefly, or while actually recording. For Google Meet or anything else, just open the app and hit Record.
    
          
          
    

          
    
            Advanced — optional "always-on tray" mode (most people don't need this)
            
    A separate, opt-in way to keep the app resident in the tray watching for calls, instead of the cold-boot flow above. This one does use memory continuously. Leave these off if you're using the switch above.
